数据结构与算法学习笔记（1）

 · 2020-3-1 · 次阅读

1.绪论

1.1 计算和算法

计算=信息处理 借助某种工具，遵照一定规则，以明确而机械的形式进行

计算模型=计算机=信息处理工具

算法(Algorithm)，即在特定计算模型下，旨在解决特定问题的指令序列。

一个算法应当具有以下基本特征：

  • 输入 待处理的信息（问题）

  • 输出 经处理的信息（答案）

  • 正确性 的确可以解决指定问题

  • 确定性 任一算法都可以描述为一个由基本操作组成的序列

  • 可行性 每一基本操作都可实现，且在常数时间内完成

  • 有穷性 对于任何输入，经有穷次基本操作，都可以得到输出

以有穷性为例，考察希尔顿序列(Hailstone Sequence)
$$ Hailstone(n)= \begin{cases} \text{{1}}&\text{$n\le1$}\\ \text{{0} $\cup$ $Hailstone(n/2)$}&\text{$n$为偶数}\\ \text{{0} $\cup$ $Hailstone(3n+1)$}&\text{$n$为奇数} \end{cases} $$
我们可以设计如下的程序，来计算给定n的Hailstone序列的长度

int hailstone(int n){
   int length = 1; //从1开始，按定义递推，累计步数，直至n=1
   while (1<n) {(n%2)? n=3*n+1 : n/=2; length++;}
   return length;
}

上述过程是否是有穷的？即对任意n，是否有：
$$
|Hailstone(n)|<\infty
$$
目前Hailstone序列还未被证明是否有穷，所以它未必是一个算法。

1.2 好算法

一个好的算法应当具有以下几个特征

  • 正确 符合语法，能够编译、链接，能够正确处理任意合法的输入
  • 健壮 能辨别不合法的输入并做适当处理，而不致非正常退出
  • 可读 结构化 + 准确命名 + 注释 +…
  • 效率 速度尽可能块；存储空间尽可能少
